about=SmartModeler GIS is a QGIS 4-only visual Processing studio. It discovers installed algorithms, builds and runs typed acyclic workflows, imports and exports native .model3 models, and offers schema-validated iterative AI planning through local or remote providers. A separate Agent Workspace dock adds bounded read-only inspections and a supervised assistant: it can propose a workflow patch, a symbology intent, a run of one reviewed core algorithm, or a run of your current workflow, but nothing takes effect until you click Apply or Run yourself, results go to temporary layers, and the last action can be undone. The assistant never invokes a plugin, runs an arbitrary algorithm, writes a file or database, or continues on its own. Only metadata leaves your machine, never feature values, source paths or credentials. API keys use password-free session memory or the optional encrypted QGIS vault. No pip dependencies are required.
